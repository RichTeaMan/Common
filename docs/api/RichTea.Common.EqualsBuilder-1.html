Append<TProperty>(Expression<Func<T, TProperty>>)

Appends an expression to the equals builder. This expression is then used on both left and right values to find a field value to be compared.

This method can handle expressions that return nulls and collections. Note that this method is considerably slower than the Append(object, object) override. That method should be used if higher performance is required,
